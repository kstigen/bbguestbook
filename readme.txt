== Description ==
Simple plugin that inserts a guestbook on a chosen page. All messages are saved to a DB-table

== Installation ==
1. Upload folder `bbguestbook` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('plugin_name_hook'); ?>` in your templates TODO
